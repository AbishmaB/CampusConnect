# 🚀 CampusConnect

CampusConnect is a full-stack social media web application built using React, Node.js, Express.js, and MongoDB.

## ✨ Features

- Create Posts
- Like Posts
- Delete Posts
- Search Posts
- Trending Section
- User Profile Page
- Responsive UI
- Modern Design

## 🛠 Tech Stack

Frontend
- React
- Vite
- Axios

Backend
- Node.js
- Express.js
- MongoDB
